## API

### Properties

| 参数       | 说明             | 类型          | 默认值 |
| ---------- | ---------------- | ------------- | ------ |
| colNumber       | 每行显示个数，受控         | number        | --      |
| defaultColNumber       | 默认每行显示个数         | number        | 3      |
| data | 文件数据 | string        | --      |
| onClick    | 图片点击事件     | function       | - -     |

## data Props

| 参数       | 说明             | 类型          | 默认值 |
| ---------- | ---------------- | ------------- | ------ |
| fileId       | 文件id         | string        | --      |
| name       | 文件名         | string        | --      |
| url | 文件链接地址 | string        | --      |
| description    | 文件描述     | string       | --      |
| isPerson  | 是否文件产生人信息         | boolean | --      |
| personInformation    | 文件产生人信息     | object       | --      |
| fileFormat  | 文件格式         | string | --      |
